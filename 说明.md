# 项目概述
## 技术栈
- **前端框架**: Vue3 + TypeScript + Element Plus
- **状态管理**: Pinia
- **构建工具**: Vite
- **桌面框架**: Tauri
- **后端语言**: Rust

## 功能
1、**跨平台支持**: Windows、MacOS、Linux全平台支持
2、查看docs文件夹下的文件 1108设计.md 1108设计2.md
